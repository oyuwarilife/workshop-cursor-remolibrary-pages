Cursor って何?

Cursorは、 AI と対話しながら作業ができるエディタです。

普通の AI ツールとの違い

普通の AI（ChatGPT 等）Cursor
会話チャットで会話するチャットで会話する
結果の扱い回答をコピペするファイルを直接作成・編集してくれる
進め方1 つずつ質問する複数ファイルを見ながら作業できる

準備手順

1 remo-library-skill をダウンロード

この手順書の「はじめに」→ 使用するファイルから remo-library-skill.zip をダウンロードしてください（このページと同じ階層に ZIP があります）。

入手できない場合は、GitHub のリポジトリから ZIP でも構いません：github.com/oyuwarilife/remo-library-skill の Code → Download ZIP。解凍後のフォルダ名が長いときは remo-library-skill にリネームしても大丈夫です。

2 ZIP を解凍する

  • Windows：ZIP を右クリック → すべて展開
  • Mac：ZIP をダブルクリック

解凍したフォルダは、デスクトップなどすぐ分かる場所に置くと当日スムーズです。

3 Cursor でフォルダを開く

  1. Cursor を起動する
  2. メニュー File → Open Folder（日本語 UI なら ファイル → フォルダを開く）
  3. 解凍した remo-library-skill フォルダを選択（ZIP ではなく、フォルダ本体）
  4. 開く をクリック

重要：ZIP のままではなく、必ず解凍したフォルダを開いてください。

4 準備できたか確認する

左のサイドバー（ファイル一覧）に、次のような名前が見えていれば OK です。

  • SKILL.md
  • README.md
  • フォルダ例：rules／categories／samples／references など
  • （あれば）preview.html や output

ここまでできていれば、当日はこのウィンドウをそのまま使えます。

症状確認すること
フォルダが開けない解凍済みか。親フォルダではなく remo-library-skill 本体を開いているか
SKILL.md が見えない別のプロジェクトを開いていないか。もう一度 Open Folder で選び直す

Cursorライティングのいまとこれから

ざっくりイメージ

📎 STEP 1 素材を渡す 記事・画像・スライドなど
＋
📐 STEP 2 ルールを足す スキル・チェック・言い回し
→
📈 RESULT 精度アップ 手直しが減っていく

ここでは、リモラボ／リモライブラリー側で試している Cursor とスキルの運用を、当日のハンズオンとは別にざっくり整理します。数字や体制は変わり得るので、「全体の空気感」の把握にお使いください。

OKいまの状態うまくいっていること

  • 1記事あたり、従来に比べて2〜3時間分くらいの作業を短くできることがある
  • ライター向けのフィードバック動画を毎回作らなくて済む場面が増えた
  • 過去記事・画像・Canva・スライドなどを読ませると、一気にそろった案が出ることがある
  • 段階を重ねて読み手を意識するチェック（いわゆるペルソナチェックなど）をスキルに載せられる
  • スキルを Claude／Cursor で管理し、スマホからも同じ流れを使いやすい体制が整いつつある

注意いまの課題ばらつき・前提

  • 一発でそのまま公開はまだ期待しすぎない方がよい（当たり外れがある）
  • 渡す情報が薄いと、内容が薄く省略されやすい
  • 過去記事の情報だけでは深掘りしきれないこともある
  • AIらしい言い回しは混じるので、人の手で整える前提が必要
  • 新企画の記事は、情報収集そのものに時間がかかりやすい
  • 環境によっては文字化けなどもゼロではない

  • クラスごとの読者像・悩みをスキルに反映
  • アンケートの生の声で冒頭の興味づけを改善
  • 使わない／使う言い回しを一覧化
  • Cursor が増やした内容の棚卸し・整理

✏️ライティングの型

  • 目的・伝えたいこと・読者の Before/Afterを先にそろえる
  • 足りなければ AI に聞き返す流れ
  • 修正をルールとして蓄積するサイクル

🗂️情報の置き場

  • リモライブラリー素材を探しやすい一箇所に
  • 情報の集め方そのものの見直し

目安のマイルストーン例

  2. 5月末まで 「ほぼ手直しなしで使える」に近い状態を約8割目安で目指す、など

※ 日程・数値はチームで合意のうえ調整してください。
