【5】推しコンテンツの推しポイントや作っていて大変なこと

記事添削会では、一人ひとりの記事を丁寧に読んで、その人に合ったアドバイスを心がけています。「ここが良い！」というポジティブなフィードバックを必ず入れながら、改善点も具体的に伝えるようにしています。大変なのは、添削に時間がかかること（笑）。一つの記事に30分以上かけることもありますが、「わかりやすかったです！」「すぐに修正できました！」という声をいただくと、疲れも吹き飛びます。ライティングって、ちょっとしたコツで劇的に変わるんです。そのコツを伝えられる瞬間が、講師としての一番のやりがいです。

Cursor を知る

イメージ

一般的な AI ブラウザでチャット → 回答をコピーして使う
Cursor エディタでチャット → フォルダ内のファイルを直接編集

Cursor って何?

Cursorは、 AI と対話しながら作業ができるエディタです。

普通の AI ツールとの違い

普通の AI（ChatGPT 等）Cursor
会話チャットで会話するチャットで会話する
結果の扱い回答をコピペするファイルを直接作成・編集してくれる
進め方1 つずつ質問する複数ファイルを見ながら作業できる

Cursor でできることの例

  1. 1
    コード作成・編集

    「このコードにエラーチェック機能を追加して」

    → AI がコードを書いてくれる

  2. 2
    文章作成

    「ブログ記事を書いて」

    → AI が記事を作成してくれる

  3. 3
    データ分析

    「この CSV データを分析して」

    → AI が分析結果をまとめてくれる

  4. 4
    ファイル整理

    「このフォルダを整理して」

    → AI がファイルを整理してくれる

  5. 5
    翻訳・要約

    「この英語の文章を日本語にして」

    → AI が翻訳してくれる

記事作成の手順

手順

Cursor のチャットを開く
Windows: Ctrl + L
Mac: Cmd + L
画面右側にチャット欄が開きます。

チャットで情報を渡す

やること: Cursor のチャット欄に、ファイルと指示とデータを一緒に
入力します。
チャットに入力する内容:
@SKILL.md
この内容に従って、リモラボの裏側の記事を書いてください
(ここに別途配布されたインタビューデータを貼り付け)

入力手順:

1. チャット欄をクリックしてカーソルを置く
2. @ を入力 → 候補から SKILL.md を選択(クリック)
3. 改行( Enter )
4. 「この内容に従って、リモラボの裏側の記事を書いてください」を
コピペ
5. 改行( Enter )を 2 回
6. 別途配布されたインタビューデータをコピペ
7. Enter キーで送信

完成イメージ:

この内容に従って、リモラボの裏側の記事を書いてください

困ったときは

🌿 つまずいても大丈夫です。まずは画面をそのまま見せる・少し待つが近道です。

困ったときは

QCursor のチャットが開かない

Aキーボードショートカット( Ctrl+L / Cmd+L )を再度押してくださ
い。または、上のメニューから開けます。

QAI が反応しない

A少し待ってみてください。生成に時間がかかることがあります。

Q: エラーメッセージが出た

Qエラーメッセージが出た

Aそのまま画面を見せてください。

Qインタビューデータがどこにあるかわからない

A別途配布されたファイルを確認してください。わからなければその
場で声をかけてください。

Q: 記事が期待と違う内容になった

Q記事が期待と違う内容になった

A大丈夫です!これも学びです。追加で指示を出して修正してみまし
ょう。わからなければ声をかけてください。

Q生成が遅い

ACursor の無料プランは速度制限があります。少し待つか、他の人の
進捗を見ながら待ちましょう。

ワークショップ後にできること

1. 別のカテゴリの記事を作る
今日の体験を活かして、以下にも挑戦してみてください:
おすすめ:
ツール紹介 - 自分が使っているツールを紹介
成⻑シェア - 自分が乗り越えた壁について書く

やり方:

@SKILL.md
この内容に従って、ツール紹介の記事を書いてください
ツール名:(例) Notion
紹介したい内容:(例)タスク管理に便利
と入力するだけで OK !

2. 記事を修正して記録をつける

生成された記事を修正したら、記録を残すと次回から精度が上がりま
す。
例:
修正記録をつけてください。
- 修正箇所 : 導入部分
- 修正前 : 「皆さま」
- 修正後 : 「みなさん」
- なぜ直したか : リモラボでは「皆さま」は使わないから

3. 自分の業務に応用する

Cursor は記事作成以外にも使えます:
メールの下書き
データの分析
議事録の整理
チェックリストの作成
情報のリサーチ
困ったことがあれば、その場で遠慮なく声をかけてください!
