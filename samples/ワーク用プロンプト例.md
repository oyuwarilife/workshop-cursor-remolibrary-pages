# ワーク用プロンプト例（Cursor チャット用）

`remo-library-skill` フォルダを Cursor で開いた状態で、**チャット（Composer / Agent）**に次を貼り付けて試せます。

ファイル参照 `@` は、実際に skill フォルダ内にサンプルをコピーした場合の例です。コピーしていない場合は、[[講師はなこ_ダミーデータ_リモラボの裏側]]の本文をチャットにそのまま貼り付けてもOKです。

---

## コピペ用

```
@SKILL.md

この内容に従って、「リモラボの裏側」の記事を書いてください。

（ここにインタビューデータを貼る。または @講師はなこ_ダミーデータ_リモラボの裏側.md を同フォルダに置いて参照）

【対象者】はなこ

【1】今、リモラボでどんなお仕事をしていますか？

（以下、インタビュー全文）
```

インタビュー全文は [[講師はなこ_ダミーデータ_リモラボの裏側]] からコピーしてください。
